Python Module and Package Syntax Examples

Module: math

The math module provides mathematical functions and constants.

  • import math
  • print(math.pi)
  • print(math.sqrt(16))

Module: os

The os module provides a portable way of using operating system-dependent functionality.

  • import os
  • print(os.getcwd())
  • print(os.listdir())

Module: random

The random module implements pseudo-random number generators for various distributions.

  • import random
  • print(random.randint(1, 10))
  • print(random.choice(['apple', 'banana', 'cherry']))

Package: numpy

NumPy is the fundamental package for scientific computing with Python.

  • import numpy as np
  • arr = np.array([1, 2, 3])
  • print(arr)

Package: pandas

Pandas is a fast, powerful, flexible, and easy-to-use open-source data analysis and manipulation library built on top of the Python programming language.

  • import pandas as pd
  • df = pd.DataFrame({'A': [1, 2], 'B': [3, 4]})
  • print(df)

Package: matplotlib

Matplotlib is a package for data visualization with Python.

  • import matplotlib.pyplot as plt
  • x = [1, 2, 3, 4, 5]
  • y = [10, 20, 25, 30, 35]
  • plt.plot(x, y)
  • plt.xlabel('x-axis')
  • plt.ylabel('y-axis')
  • plt.title('Title')
  • plt.show()

Package: scikit-learn

Scikit-learn is a machine learning library for Python.

  • from sklearn import datasets
  • from sklearn.model_selection import train_test_split
  • from sklearn.neighbors import KNeighborsClassifier
  • iris = datasets.load_iris()
  • X_train, X_test, y_train, y_test = train_test_split(iris.data, iris.target, test_size=0.3)
  • knn = KNeighborsClassifier(n_neighbors=3)
  • knn.fit(X_train, y_train)
  • print(knn.score(X_test, y_test))

Package: seaborn

Seaborn is a Python data visualization library based on matplotlib.

  • import seaborn as sns
  • tips = sns.load_dataset("tips")
  • sns.histplot(data=tips, x="total_bill", hue="sex", kde=True)
