Django Project Creation Steps

Step 1: Install Django

Use pip to install Django framework.

  • pip install django

Step 2: Create Django Project

Use django-admin to start a new Django project.

  • django-admin startproject <projectname>

Step 3: Navigate to Project Directory

Change directory to the newly created Django project.

  • cd <projectname>

Step 4: Create Django App

Use manage.py to start a new Django app within the project.

  • python manage.py startapp <appname>

Step 5: Run Django Development Server

Start the Django development server to run the project.

  • python manage.py runserver
